## Run Locally

Clone the project

```bash
  git clone https://github.com/kiruba56/rn-omegle
```

Go to the project directory

```bash
  cd rn-omegle
```

Install dependencies

```bash
  yarn install
```

**Android**

```bash
  yarn run android
```

**iOS**

```bash
  pod install --project-directory=ios && yarn run ios
```
